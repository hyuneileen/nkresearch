NaI(Tl)섬광검출기 는 γ선에 대한 감도가 매우 높은것으로 하여 스펙트르분석기와 선량계용검출기로 널리 쓰 인다. 그러나 NaI(Tl)섬광검출기는 γ선의 에네르기에 따라 기록효률이 심히 차이나는것으로 하여 선량측정에서 일련의 문제점들이 나선다. NaI(Tl)섬광검출기의 에네르기에 따르는 흡수 선량감도의 차이를 해결하기 위한 방법으로서 금속함차페법과 회로보상에 의한 방법 등이 제기되였으나 에네르기의 전구간에서 선량감도의 차이를 완전히 해결하지 못하였다. 현재 세계적으로 스펙트르를 측정하여 흡수선량을 평가하기 위한 연구사업들이 많이 진행되고있으나 측정방법의 일반적인 원리만 밝히고 구체적인 자료는 밝히지 않고있다.[1, 2] 우리는 한소편콤퓨터기술과 콤퓨터모의기술이 높은 수준에로 발전한 현실의 요구에 맞게 스펙트르측정에 의한 흡수선량평가방법을 제기하고 그에 대한 연구를 진행하였다. １．스펙트르측정에 기초한 흡수선량평가방법에 대한 리론적고찰 스펙트르에 의한 흡수선량평가는 주로 두가지 방향으로 진행되는데 하나는 진폭스펙 (EG 함 ) 트르를 분석하지 않고 직접 스펙트르－선량변환연산자를 리용하여 평가하는 방법( 수방법이라고 한다.)이고 다른 하나는 진폭스펙트르를 분석하여 입사에네르기와 흐름을 결정한 다음 선량을 평가하는 방법(입사흐름평가에 의한 방법)이다. ) (EG 함수방법 에네르기가 0E 인 γ량자흐름이 검출기에 수직으로 입사하는 경우 흐름 으로, 흡수선량을 의 세기를 단위세기로 보고 검출기에 기록된 스펙트르밀도를 ( 0ED 라고 하면 다음과 같은 식이 얻어진다. 0EEn , ( ) ) ED ( ) = 0 dEEGEEn ) ⋅ ( ) ( , (1) 0 이와 같이 정의된 (EG 함수를 스펙트르－선량변환연산자라고 한다. ) E 통로에 대한 기록수는 다음과 같이 표시된다.
